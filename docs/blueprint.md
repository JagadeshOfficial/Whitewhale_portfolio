# **App Name**: WhiteWhale Digital HQ

## Core Features:

- Dynamic Service Showcase: Showcase the company's services like IT Outsourcing, Software Development, and Venture Capital with dynamically rendered content.
- Client Logo Display: Visually showcase client logos to reinforce credibility.
- Careers Portal: Frontend for a careers page displaying job openings and application form. Note: this MVP only covers the application form creation and submissions, not review and workflow.
- ATS-Friendly Resume tool: A tool for generating resumes using AI, which is capable of considering which details might be useful, and when to include them based on user selections. It provides suggestions for content, optimized for Applicant Tracking Systems.

## Style Guidelines:

- Primary color: Deep teal (#008080) reflecting trustworthiness and innovation, inspired by the consulting firm's focus on investment and technology.
- Background color: Very light grayish-teal (#F0F8FF) for a clean and professional feel.
- Accent color: Blue-green (#468499) to highlight key sections and CTAs, analogous to the primary, ensuring a consistent feel, yet significantly different in saturation.
- Body and headline font: 'Inter', a sans-serif font with a modern and neutral appearance, suitable for both headlines and body text.
- Responsive layout based on Tailwind CSS, optimized for various screen sizes.
- Use a consistent set of professional icons to enhance visual communication.
- Subtle transitions and animations to improve user engagement and provide visual feedback.
